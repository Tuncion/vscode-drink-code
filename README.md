> Stay on top of your hydration game while you code! "Drink Code" helps developers maintain their health by sending gentle reminders to drink water during coding sessions.

### 🏠 [Homepage](https://tuncion.de)

## Prerequisites

- vscode ^1.96.0

## Install

```sh
npm install
```
